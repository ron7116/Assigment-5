Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the Document Object Model (DOM): 1.By Id 2.By ClassName 3.By Tag Name 4.By CSS Selector 5.By Name Attribute 6.By Traversal Methods(like: parentNode, childNode etc.).

Question-2: What is event bubbling in the DOM?

Event bubbling is a process where an event that is triggered on a child element first runs its associated event handlers, then “bubbles up” the DOM tree to its parent elements, triggering their event handlers, and continues to bubble up until it reaches the root element.

Question-3: What is the difference between innerHTML, innerText, and textContent ?

innerText : innerText get or sets the text content within an HTML element, but it only includes visible text.It ignores HTML tags. textContent : textContent retrieves or sets the text within an HTML element. it retrieves all text, even if it’s hidden or styled not to display. innerHtml : innerHTML is a bit different from both innerText and textContent. Instead of just the text, innerHTML returns or sets the HTML markup within an element.

Question-4: What is event delegation in the DOM?

Where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements, instead of adding separate listeners to each child element

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create Element : To create a new HTML element, use the document.createElement method. 2. Add Element : When you've created an element, you can add it to the DOM using methods like appendChild or append. 3. Remove Element : To remove an element from the DOM, you can use methods like removeChild or the remove method.
